## Viewing Support Requests  

The support section provides an overview of all customer support requests. Each request includes the customer's email, subject, and timestamp, making it easy to track and manage responses.  

<img src="/images/support/email-support-list.png" alt="Viewing support requests in Laxi.ai" style={{ marginTop: '1rem', maxWidth: '100%' }} />  

Click on any support request to view its details and customer message.

## Filtering Support Requests  

To manage inquiries efficiently, Laxi.ai provides filtering options that help you sort through requests.  

### By Response Status  
You can filter support requests based on whether they have been replied to or are still awaiting a response. This helps prioritize urgent messages.  

<img src="/images/support/email-support-filter-replied.png" alt="Filtering support requests by response status" style={{ marginTop: '1rem', maxWidth: '50%', marginLeft: 'auto', marginRight: 'auto' }} />  

### By Date Range  
For better tracking, use the date range filter to view support requests within a specific period. This is useful for analyzing response trends over time.  

<img src="/images/support/email-support-filter-date.png" alt="Filtering support requests by date range" style={{ marginTop: '1rem', maxWidth: '100%' }} />

The Email Support section makes it easy to manage and respond to customer inquiries efficiently. By using the available filters and viewing message details, you can ensure timely and effective communication with your customers.
